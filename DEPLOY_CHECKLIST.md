# ✅ デプロイ前チェックリスト

デプロイ前に、以下の項目を確認してください。

## 📋 必須チェック項目

### 1. ファイルの確認

- [ ] `requirements.txt` が存在し、すべての依存パッケージが記載されている
- [ ] `app.py` が存在し、エラーなく実行できる
- [ ] `.gitignore` に機密情報（`data/`、`.streamlit/secrets.toml`など）が含まれている

### 2. コードの確認

- [ ] APIキーがコードに直接書かれていない
- [ ] パスワードや機密情報がコードに含まれていない
- [ ] ローカル環境で動作確認が完了している

### 3. 設定ファイルの確認

- [ ] `.streamlit/config.toml` が存在する（ローカル用）
- [ ] `.streamlit/secrets.toml.example` が存在する（参考用）

### 4. データファイルの確認

- [ ] `data/` ディレクトリが `.gitignore` に含まれている
- [ ] サンプルデータがリポジトリに含まれていない

---

## 🔍 デプロイ方法別の追加チェック

### Streamlit Cloud

- [ ] GitHubリポジトリが作成されている
- [ ] コードがGitHubにプッシュされている
- [ ] Streamlit Cloudアカウントが作成されている
- [ ] APIキーを設定する場合は、Streamlit Secretsの形式を確認

### Railway

- [ ] Railwayアカウントが作成されている
- [ ] GitHubリポジトリが接続されている
- [ ] 環境変数（`GROK_API_KEY`など）を設定する準備ができている
- [ ] 起動コマンドが正しく設定されている

### Render

- [ ] Renderアカウントが作成されている
- [ ] GitHubリポジトリが接続されている
- [ ] 環境変数を設定する準備ができている
- [ ] Build CommandとStart Commandが正しく設定されている

---

## 🧪 デプロイ後の確認項目

デプロイが完了したら、以下を確認してください：

- [ ] アプリが正常に起動している
- [ ] すべてのページが表示される
- [ ] フォームが正常に動作する
- [ ] データの保存が正常に動作する（可能な場合）
- [ ] APIキーが正しく認識されている（設定した場合）
- [ ] エラーログに問題がない

---

## 🆘 問題が発生した場合

1. **ログを確認**
   - Streamlit Cloud: デプロイ画面の「Logs」タブ
   - Railway: 「Deployments」タブのログ
   - Render: 「Logs」タブ

2. **よくある問題**
   - `requirements.txt` にパッケージが不足している
   - ポート番号の設定が間違っている
   - 環境変数が正しく設定されていない

3. **解決方法**
   - エラーメッセージを確認
   - `DEPLOY.md` のトラブルシューティングセクションを参照
   - 必要に応じて、ローカル環境で再テスト
